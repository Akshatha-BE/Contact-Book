PROJECT DESCRIPTION	

This project is a simple Contact Book Web Application developed using Python Flask.
It allows users to manage their contacts by adding, viewing, editing, and deleting contact details through a web interface.

The application stores contact information in a JSON file and provides a simple and user-friendly interface for managing contacts efficiently.

FEATURES

Add new contacts
View saved contacts
Edit contact information
Delete contacts
Simple and responsive user interface

TECHNOLOGIES USED

Python
Flask
HTML
CSS
JSON

CONCLUSION

This mini project demonstrates the basic implementation of a Flask web application for managing contact information. It helps in understanding CRUD operations, Flask routing, templates, and simple data storage using JSON.
